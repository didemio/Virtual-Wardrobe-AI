# Virtual Wardrobe AI

This is an AI-powered virtual wardrobe application that helps users manage their clothes and get outfit suggestions.

## Features

- Load wardrobe data from JSON
- Suggest outfits based on categories
- AI agent for interactions

## Installation

1. Install dependencies: `pip install -r requirements.txt`
2. Run the agent: `python agent.py`

## Project Structure

- `agent.py`: Main AI agent
- `tools/wardrobe_tools.py`: Tools for wardrobe management
- `data/wardrobe.json`: Sample wardrobe data
- `tests/`: Unit tests
